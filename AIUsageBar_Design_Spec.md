# AIUsageBar — Дизайн-спецификация (iOS 18 style)

Документ для передачи в Claude Code. Описывает визуальный редизайн menubar-приложения AIUsageBar. Цель — осовременить интерфейс в стиле iOS 18: цветные карточки, фирменные оттенки сервисов, скругления, мягкие фоны, иконки.

Текущий стек: SwiftUI, macOS 13+. Менять архитектуру/логику не нужно — только визуальный слой (ContentView.swift и связанные view-структуры).

---

## 1. Общие принципы

- **Стиль:** iOS 18 / macOS Sonoma — мягкие карточки, крупные скругления, заметные но не кричащие цвета.
- **Каждый бар — в своём смысловом цвете** (не один цвет на всё).
- **Фирменный акцент на вкладку:** Claude = терракота, ChatGPT = зелёный.
- **Плотность:** воздуха больше, чем сейчас. Карточки с внутренними отступами 14px, между карточками 14px.
- **Без градиентов-переливов** на барах — чистые сплошные заливки (можно лёгкий вертикальный градиент 1 тон, опционально).
- **Тёмная и светлая тема** — обе обязательны. Использовать системные цвета и opacity, чтобы адаптировалось автоматически.

### Состояния заполнения (перекрывают цвет бара)
- **> 80% использовано** → бар становится оранжево-красным `#E8743B` / системный `.orange`, процент тоже.
- **> 95%** → красный `#E24B4A` / системный `.red`.
- Иначе — фирменный цвет метрики.

### Фоны и текст (адаптивные)
- Фон попапа: `Color(NSColor.windowBackgroundColor)`
- Фон карточки: `Color(NSColor.controlBackgroundColor)` или `Color.primary.opacity(0.04)`
- Трек прогресс-бара (пустая часть): `Color.primary.opacity(0.10)`
- Заголовок метрики: `.primary`, 14pt, `.semibold`
- Подпись (Resets…): `.secondary` → лучше `Color.secondary.opacity(0.8)`, 12pt
- Процент: 13pt, `.semibold`, в цвет бара

## 3. Размеры и геометрия

- Ширина попапа: **340px** (как сейчас), высота **460px**.
- Радиусы скруглений:
  - Карточка метрики: **16px**
  - Иконка сервиса (квадрат в шапке): **9px**
  - Прогресс-бар: высота **9px**, радиус **5px** (полностью скруглённые концы)
  - Кнопки/пилюли вкладок: **11px**
- Внутренний отступ карточки: **14px** со всех сторон.
- Отступ между карточками: **14px**.
- Горизонтальные поля контента: **18px**.

## 5. Детали компонентов

### 5.1 Шапка (header)
- Слева: цветной квадрат-иконка **30×30px**, радиус 9px, заливка = фирменный цвет активного сервиса, внутри белая SF Symbol (`sparkles` для Claude, `bubble.left.fill` для ChatGPT), размер символа 17pt.
- Рядом название активного сервиса: 18pt, `.medium`, `.primary`.
- Справа: кнопка обновления — SF Symbol `arrow.clockwise`, 17pt, цвет `.secondary`. При клике — лёгкая анимация поворота на 360° (`rotationEffect` + `withAnimation`).
- Отступы: padding сверху 18, по бокам 18, снизу 14.

### 5.2 Вкладки (segmented pills)
- Две пилюли в ряд, каждая `flex` на половину ширины.
- Активная: заливка = фирменный цвет сервиса, текст белый, 13pt `.medium`.
- Неактивная: заливка `Color.primary.opacity(0.06)`, текст `.secondary`.
- Радиус 11px, padding по вертикали 7px.
- Анимация перехода между вкладками: `.easeInOut(duration: 0.2)`.

### 5.3 Карточка метрики
Это главный переиспользуемый компонент. Структура внутри карточки:

1. **Верхняя строка** (HStack, space-between):
   - Слева: название метрики, 14pt `.semibold` `.primary`.
   - Справа: процент «12%», 13pt `.semibold`, цвет = цвет бара (или оранжевый/красный при высоком заполнении).
2. **Прогресс-бар** (отступ сверху 10px):
   - Высота 9px, радиус 5px.
   - Трек: `Color.primary.opacity(0.10)`.
   - Заполнение: ширина = `geo.size.width * fraction`, цвет бара, радиус 5px.
   - **Анимация:** при появлении/обновлении бар «доезжает» от 0 до значения, `.spring(response: 0.6, dampingFraction: 0.8)`.
3. **Строка сброса** (отступ сверху 9px):
   - SF Symbol `clock`, 13pt, `.tertiary`.
   - Текст «Сброс 25 июня, 3:29», 12pt, `Color.secondary.opacity(0.8)`.

Фон карточки: `Color.primary.opacity(0.04)` (адаптивно), радиус 16px, padding 14px.

### 5.4 Футер
- Тонкий разделитель сверху (`Divider`, opacity 0.5).
- Три элемента в ряд: «🔍 Debug» (раскрывашка, слева), «Обновляется каждые 15 мин» (центр, 11pt `.secondary`), «Выйти» (справа, 11pt `.secondary`, кнопка).
- Padding по бокам 16, по вертикали 10.
- Debug-блок при раскрытии — моноширинный текст 9pt в карточке `Color.primary.opacity(0.04)`.

## 6. Состояния (states)

### 6.1 Не залогинен
По центру, вертикально:
- Цветной круг **64×64px** заливка = фирменный цвет с opacity 0.15, внутри SF Symbol сервиса в фирменном цвете, 28pt.
- Заголовок «Войдите в Claude», 15pt `.medium`.
- Подзаголовок «Для отображения лимитов нужна авторизация», 12pt `.secondary`, по центру.
- Кнопка «Войти в Claude» — заливка фирменным цветом, белый текст, радиус 11px, padding 10×20. Используй `.buttonStyle(.plain)` + кастомный фон, чтобы цвет был фирменный, а не системный синий.

### 6.2 Загрузка
- По центру `ProgressView()` + «Загрузка…», 13pt `.secondary`.

### 6.3 Нет данных / безлимит
- SF Symbol `checkmark.seal.fill` 28pt в зелёном, «Лимиты не найдены», подпись «Возможно, у вас безлимитный план».

### 6.4 Высокое заполнение (предупреждение)
- При проценте > 80: бар и процент оранжевые, рядом с процентом маленький SF Symbol `exclamationmark.triangle.fill` 11pt.
- При > 95: красный.

## 7. Анимации (суммарно)

| Элемент | Анимация |
|---------|----------|
| Прогресс-бар при загрузке | `.spring(response: 0.6, dampingFraction: 0.8)` от 0 до значения |

| Кнопка refresh | поворот 360° за 0.5s при клике |
| Появление карточек | fade + slide снизу на 8px, stagger по 0.05s |

## 9. Что НЕ менять

- Логику получения данных (UsageStore, WKWebView, JS-запросы) — не трогать.
- Структуру данных `ServiceUsage`, `UsageBar` — оставить как есть (поля `label`, `percent`, `resetAt`).
- Menubar-иконку (отдельная монохромная) — оставить.
- Поведение логина (LoginWindowController) — оставить.

Менять только: `ContentView.swift` и вложенные view-структуры (`ClaudeView`, `ChatGPTView`, `UsageBarRow`, `SimpleUsageRow`, `NotLoggedInView`, и т.п.) — их визуальную часть.
